# 此工程机 P0 / P1 / P2 开发排期版计划

## P0
- 运行时最小下单量 / 最小名义金额校验。
- 扫描阶段改用扣费后的净 APR。
- 单币 / 单所 / 总暴露 / 总回撤限制。
- 启动只读健康检查。
- 残仓、平仓失败、恢复异常的结构化告警。

## P1
- 真正落地盘口流动性与深度过滤。
- 各交易所 market metadata 继续补齐与统一。
- dashboard 增加 close reason、净收益、风险状态。
- funding history 与对账能力增强。

## P2
- 分币种参数体系。
- 交易所可靠性评分与自动降级。
- 资金再平衡、暂停单所、标准化 runbook 演练。
